Smart Home

❮ ❯

This smarthome is made using a Wemos D1 R2 microcontroller and a relay module to control the on/off of lamps, water pumps and other electronic devices. This system was created using the C++ programming language, and so that it can be controlled remotely this system is connected to the Firebase real time database. The way this system works is that when the toggle on the application is turned on, the application will send command 1 to Firebase which will be received by the microcontroller and the relay will respond to the command.
